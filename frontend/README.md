# SISIG POS Frontend

React + Ant Design + SASS frontend for the SISIG POS + Inventory System.

Quick start

1. Install dependencies

```bash
cd frontend
npm install
```

2. Run the dev server

```bash
npm start
```

API endpoints expected (backend to be created later):
- `GET /api/menu/` — fetch menu items
- `POST /api/orders/create` — create order
- `GET /api/inventory/` — get inventory
